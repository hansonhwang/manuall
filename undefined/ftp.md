# 마이 FTP

## 관리자 메뉴 위치

`MY메뉴` > `마이 FTP`

## 메뉴 안내 및 설정 방법

**마이 FTP 이미지 저장 공간**

<figure><img src="../.gitbook/assets/image (5).png" alt=""><figcaption></figcaption></figure>

**① 이미지 저장 공간**

* 상품 상세 설명 등에 사용할 이미지를 저장하는 공간이에요.
* 기본으로 1GB의 저장 용량이 무료로 제공돼요. 화면 상단에서 사용 중/남은 용량을 확인할 수 있어요.

<figure><img src="../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>

**② 이미지 업로드**

* 내 컴퓨터의 이미지를 등록하면 저장 공간에 자동으로 저장돼요. 공간이 부족하면 더 이상 업로드할 수 없어요.

<figure><img src="../.gitbook/assets/image (7).png" alt=""><figcaption></figcaption></figure>

**③ 파일 불러오기**

* 상품 등록 또는수정시에 이미 저장된 이미지를 선택해 다시 사용할 수 있어요.

{% hint style="warning" %}
참고

저장 공간이 부족하거나 오픈마켓에서도 이미지를 사용하려면 `이미지호스팅`으로 기본 저장소를 변경하세요. 용량 제한 없이 사용할 수 있어요.
{% endhint %}
