# 회원 상세

## 관리자 메뉴 위치

`회원` > `회원 목록` > `이름 또는 아이디 클릭`

## 메뉴 안내 및 설정 방법

{% tabs %}
{% tab title="개인 정보" %}
<figure><img src="../.gitbook/assets/Wireframe (1) (1).png" alt=""><figcaption></figcaption></figure>

**① 기본 정보**

* **아이디** : 회원 아이디로, 수정할 수 없어요.
* **이름** (필수) : 최대 30자까지 입력할 수 있어요.
* **이메일** (필수) : 최대 35자. 형식이 맞지 않거나 다른 회원과 중복되면 안내가 표시돼요.
* **휴대폰번호** (필수) : '-' 없이 숫자만, 10\~11자리로 입력하세요.
* **비밀번호 변경** : 새 비밀번호와 확인을 입력해 재설정해요. 두 값이 다르면 "비밀번호가 일치하지 않습니다." 안내가 표시돼요.
* **CRM 그룹** : 최우수 / 우수 / 일반 / 주의 / 항의 고객 중 선택해요.

**② 이용 정보**

* **결제 금액** : 누적 결제 금액(결제 실패·환불 제외). 누르면 이 회원의 `주문 목록`이 새 창으로 열려요.
* **상품 문의** : 누적 문의 건수. 누르면 이 회원의 `상품 문의` 내역으로 이동해요.
* **상품 리뷰** : 누적 리뷰 건수. 누르면 이 회원의 `상품 리뷰` 내역으로 이동해요.
* **최근 접속일** : 마지막 접속 일시와 IP. (접속 이력이 없으면 가입일과 동일하게 표시)
* **가입일** : 회원 가입 일시

**③ 회원 메모**

* 회원에 대해 관리자만 볼 수 있는 메모를 남길 수 있어요. (국문 최대 100자, 영문 최대 200자)

**④ 저장**

* 수정한 뒤 우측 상단의 `저장` 버튼을 누르세요. 저장 후에도 현재 화면이 유지돼요.
