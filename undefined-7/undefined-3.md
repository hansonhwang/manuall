# 프리미엄 업그레이드

## 관리자 메뉴 위치

`부가서비스` > `프리미엄 업그레이드`

## 메뉴 안내 및 설정 방법

프리미엄 업그레이드 안내 화면

1. **업그레이드 소개**
   * Lite의 제한(예: 상품 최대 50개 등)을 넘어 더 많은 기능을 쓰고 싶을 때 프리미엄으로 전환하는 서비스예요.
2. **요금·기능 안내**
   * 프리미엄에서 제공되는 추가 기능과 요금을 확인할 수 있어요.
3. **신청**
   * 안내에 따라 신청하면 프리미엄 기능을 이용할 수 있어요.

{% hint style="warning" %}
참고

상품 등록 한도에 도달했거나 더 다양한 분류·기능이 필요할 때 업그레이드 안내가 표시돼요.
{% endhint %}
