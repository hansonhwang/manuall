# SMS 충전

## 관리자 메뉴 위치

`부가서비스` > `SMS 충전`

## 메뉴 안내 및 설정 방법

**① 잔액 확인**

* 현재 남은 문자 발송 잔액을 확인할 수 있어요.

**② 충전 금액 선택·결제**

* 충전할 금액을 선택하고 결제하면 잔액이 채워져요.

**③ 충전·사용 내역**

* 충전 일시와 문자 발송으로 차감된 내역을 확인할 수 있어요.

{% hint style="warning" %}
참고

문자 발송(`SMS` 메뉴)은 잔액에서 차감돼요. 단문(SMS)과 장문(LMS)은 차감 금액이 달라요.
{% endhint %}
