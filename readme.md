# README

메이크샵 Lite 관리자(셀러 어드민)의 사용 방법을 안내하는 가이드입니다.\
상품 등록부터 주문·회원 관리, 부가서비스, 설정까지 메뉴별로 따라 하시면 됩니다.

> 셀러 어드민 ㅣ **(어드민 URL 입력)**

***

## 이 가이드는 이렇게 구성돼 있어요

좌측 메뉴 순서와 동일하게 영역을 나누어 안내합니다.

| 영역              | 주요 내용                          |
| --------------- | ------------------------------ |
| **시작하기**        | 화면 구성, 홈(대시보드), 마이 FTP 등 기본 개념 |
| **상품**          | 상품 목록 · 등록 · 수정                |
| **주문**          | 주문 목록 · 상세                     |
| **회원**          | 회원 목록 · 상세                     |
| **문의·리뷰**       | 상품 문의 · 리뷰 · 공지사항              |
| **SMS / 자동 메일** | 고객 대상 메시지·메일 발송                |
| **통계**          | 매출·방문 등 데이터 확인                 |
| **부가서비스**       | 이미지호스팅, 택배 연동, 네이버 연동 등        |
| **설정**          | 기본 정보, 배송·결제·도메인 등 운영 설정       |

{% hint style="info" %}
처음 사용하신다면 **시작하기 > 화면 구성 안내** 부터 읽어 보시길 권장합니다.
{% endhint %}
