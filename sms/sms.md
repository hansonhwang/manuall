# SMS

## 관리자 메뉴 위치

`SMS`

## 메뉴 안내 및 설정 방법

{% stepper %}
{% step %}
## 수신자 선택

* 회원 목록에서 선택하거나, 번호를 직접 입력해 수신자를 지정해요.
{% endstep %}

{% step %}
## 메시지 작성

* 내용을 입력해요. 글자 수에 따라 단문(SMS)/장문(LMS)으로 자동 구분돼요.
* 발신번호는 미리 등록·인증한 번호 중에서 선택해요. (설정 > 발신번호 관리)
{% endstep %}

{% step %}
## 발송 · 예약

* 즉시 발송하거나, 원하는 날짜·시간으로 예약 발송할 수 있어요.
{% endstep %}

{% step %}
## 발송 내역

* 보낸 메시지의 발송 일시, 수신자, 성공/실패 결과를 확인할 수 있어요.
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
참고

SMS 발송에는 충전 잔액이 필요해요. 잔액이 부족하면 `부가서비스 > SMS 충전`에서 충전해 주세요.
{% endhint %}
