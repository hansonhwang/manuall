상품 상세페이지 생성기

상품명과 이미지를 넣으면 카테고리에 맞는 스타일로 상세페이지 이미지를 AI로 생성하고, 우측에 완성된 HTML 상세페이지를 만들어줍니다.

Google Gemini API 키 설정

아래 "이미지 생성 방식"을 레이아웃만(무료)로 두면 API 키 없이도 사용할 수 있습니다. 키를 입력하면 소개 문구는 AI로 다듬어주고, AI 이미지 생성(유료) 모드에서는 이미지도 새로 생성합니다.
Google AI Studio에서 발급. 이 키는 이 브라우저에만 저장되며 API 호출에 직접 사용됩니다(서버 경유 없음 = 네트워크 탭에 노출될 수 있으니 개인/테스트 용도로만 사용하세요).
주의: 이미지 생성 모델(gemini-2.5-flash-image)은 무료 티어 할당량이 0이라 결제(billing)가 연결된 Google Cloud 프로젝트의 키가 필요합니다. 무료 키만 있다면 429 quota exceeded 오류가 발생할 수 있습니다.

신선원물을 선택하면 조리 이미지 대신 수확한 그대로의 신선한 원물 이미지로 생성됩니다.
