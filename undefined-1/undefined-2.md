# 상품 목록

## 관리자 메뉴 위치

`상품` > `상품 목록`

## 메뉴 안내 및 설정 방법

<figure><img src="../.gitbook/assets/image (17).png" alt=""><figcaption></figcaption></figure>

{% stepper %}
{% step %}
### 공통 정보 관리 · 분류 관리 · 상품 등록

* 화면 우측 상단의 버튼으로 주요 기능에 바로 이동할 수 있어요.
* **공통 정보 관리**: 여러 상품에 공통으로 들어가는 정보(배송·교환·환불 안내 등)를 관리해요.
* **분류 관리**: 상품 분류(카테고리)를 추가·수정·삭제해요.
* **상품 등록**: 새 상품을 등록하는 화면으로 이동해요.
* 모바일에서는 `관리` 버튼 안에 공통 정보 관리 / 분류 관리가 들어 있어요.


{% endstep %}

{% step %}
### 상품 검색

* 검색 조건(상품명 / 상품번호)을 선택하고 검색어를 입력한 뒤 Enter를 누르세요.
* 검색어가 일부만 일치해도 결과에 표시되고, 비워 두고 검색하면 전체 상품이 조회돼요.
* `상세 검색`을 누르면 판매가·재고 상태 등 여러 조건을 함께 지정할 수 있어요.

<figure><img src="../.gitbook/assets/image (19).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### 검색 결과 건수 · 페이지당 개수

* 조회된 상품 수가 표시돼요.
* 한 페이지에 보여줄 상품 수를 10 / 30 / 50 / 100개 중에서 선택하세요. (기본 30개, 설정 자동 저장)
{% endstep %}

{% step %}
### 상품 리스트

* 조회된 상품이 등록일 최신순으로 표시돼요.
* 구성: 상품 정보(이미지·상품번호·상품명·분류·상품 복사 버튼), 판매가, 재고 상태, 노출 상태, 등록일(수정일)
* 상품 이미지를 누르면 쇼핑몰 상품 페이지로 이동해요. (노출 안 함 상품은 "존재하지 않는 상품입니다." 안내 후 메인으로 이동)
* 상품명을 누르면 상품 수정 화면으로 이동해요.
* 분류를 지정하지 않은 상품은 `분류 선택 안 함`으로 표시돼요.
{% endstep %}

{% step %}
### 상품 복사

* 상품 복사 버튼을 누르고 확인하면 기존 상품의 정보와 이미지가 복사되어 새 상품으로 등록돼요.
* 복사된 상품은 `노출 안 함` 상태로, 목록 맨 위에 추가돼요.
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
참고

메이크샵 Lite에서는 상품을 최대 50개까지 등록할 수 있어요. 등록 가능 수가 2개 남으면 안내 배너가 표시되고, 한도에 도달하면 상품 등록·복사가 제한돼요.
{% endhint %}
