# 상품 수정

## 관리자 메뉴 위치

`상품` > `상품 목록` > `상품명 클릭`

## 메뉴 안내 및 설정 방법

상품명·판매가·옵션·이미지 등 **입력 항목과 작성 방법은** [**상품 등록**](/broken/pages/c89c6b1de1ce818de43fca24ab4077cd3385f53c)**과 같아요.** 여기서는 상품 등록과 다른 부분만 안내해요.



**① 등록일 · 수정일**

* 화면 상단에서 상품의 등록일과 마지막 수정일을 확인할 수 있어요.

**② 상품 번호 · 상품 코드**

* 상품 등록 시 자동으로 생성되는 값으로, 수정 화면에서 확인할 수 있어요.

**③ 상품 보러가기 · 공유하기**

* **상품 보러가기** : 쇼핑몰의 해당 상품 페이지로 이동해요. (노출 안 함 상품은 "존재하지 않는 상품입니다." 안내 후 메인으로 이동)
* **공유하기** : 누르면 상품 URL이 복사되고 "상품 URL이 복사되었습니다." 안내가 표시돼요. 노출 상태와 관계없이 항상 사용할 수 있어요.

**④ 저장**

* 내용을 수정한 뒤 우측 상단의 `저장` 버튼을 누르세요. 저장 후에도 현재 화면이 유지돼 이어서 수정할 수 있어요.

**⑤ 삭제**

* 우측 상단의 `삭제` 버튼을 누르고 확인하면 상품이 삭제되고, 들어왔던 상품 목록 위치로 돌아가요.

{% hint style="warning" %}
참고

삭제한 상품은 되돌릴 수 없어요. 삭제 전에 한 번 더 확인해 주세요.
{% endhint %}
