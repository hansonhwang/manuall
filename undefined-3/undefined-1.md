# 상품 문의

## 관리자 메뉴 위치

`문의·리뷰` > `상품 문의`

## 메뉴 안내 및 설정 방법

<figure><img src="../.gitbook/assets/image (22).png" alt=""><figcaption></figcaption></figure>

**① 문의 검색·필터**

* 답변 상태(답변 대기 / 답변 완료), 상품명, 작성자 등으로 문의를 검색할 수 있어요.

**② 문의 리스트**

* 구성: 상품 정보, 문의 내용, 작성자, 작성일, 답변 상태
* 문의를 누르면 상세 내용과 답변 작성 영역이 열려요.

**③ 답변 작성**

* 고객 문의에 답변을 작성하면 상태가 `답변 완료`로 바뀌고, 쇼핑몰 상품 페이지에 함께 표시돼요.
* 비공개 문의는 작성자와 관리자만 볼 수 있어요.

{% hint style="warning" %}
참고

답변 대기 중인 문의는 빠르게 처리할수록 구매 전환에 도움이 돼요. 홈(대시보드)에서도 대기 건수를 확인할 수 있어요.
{% endhint %}
