# 주문 상세

## 관리자 메뉴 위치

`주문` > `주문 목록` > `상품 정보 클릭`

## 메뉴 안내 및 설정 방법

<figure><img src="../.gitbook/assets/image (21).png" alt=""><figcaption></figcaption></figure>

**① 주문 기본 정보**

* 상단에 주문번호와 주문일시가 표시돼요. 주문번호 옆 복사 버튼을 누르면 번호가 복사돼요.

**② 상태별 보기**

* 주문 상태별 품목 수가 표시되며, 탭을 누르면 해당 상태의 상품만 조회돼요. (전체 · 취소 · 교환 · 반품 · 환불)
* 해당 상태의 상품이 없으면 `"해당하는 상품이 없습니다."` 문구가 표시돼요.

**③ 주문 상품 처리**

* **송장번호 입력**: 배송을 시작할 때 택배사·송장번호를 입력해요.
* **배송완료 처리**: 배송이 끝난 주문을 완료 상태로 변경해요.
* **취소 · 교환 · 반품 접수**: 처리 버튼을 누르면 각 접수 창이 열려요.

**④ 취소 · 교환 · 반품 · 환불 처리**

* 상품의 현재 상태에 따라 가능한 처리만 표시되고, 단계별로 진행돼요.
* **취소**: 취소 접수 → 취소 완료 / 취소 거부
* **교환**: 교환 접수 → 수거 → 교환 완료 / 교환 불가
* **반품**: 반품 접수 → 수거 → 반품 완료 / 반품 불가
* **환불**: 환불 처리 → 환불 완료
* 처리가 완료되면 결제 금액과 상태가 자동으로 갱신돼요.

**⑤ 결제 정보**

* **최초 결제 금액**: 주문 시 처음 결제된 금액
* **총 결제 금액**: 현재 결제 금액 (환불 발생 시 환불 금액 제외)
* **상품 금액 · 배송비 · 결제 수단**: 금액 구성과 결제 수단(카드는 승인번호 표시)
* **추가 결제 금액**: 교환·반품 과정에서 추가로 발생한 금액

**⑥ 주문자 정보**

* 주문자의 이름·아이디·휴대폰·이메일이 표시돼요.
* `메시지 발송`으로 주문자에게 메시지를 보내거나, `주문 전환`을 사용할 수 있어요.

**⑦ 배송지 정보**

* 수령인, 휴대폰번호, 주소, 주문 메시지, 배송 메시지가 표시돼요. 배송 전이라면 배송지 정보를 수정할 수 있어요.

**⑧ 관리 메모**

* 주문에 대해 관리자만 볼 수 있는 메모를 남길 수 있어요. (고객에게는 보이지 않아요.)

{% hint style="warning" %}
참고

취소·교환·반품·환불은 주문의 현재 상태에 따라 가능한 처리만 표시돼요. 잘못 처리하지 않도록 상태를 확인한 뒤 진행하세요.
{% endhint %}
